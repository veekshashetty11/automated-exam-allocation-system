About the Project

This project demonstrates an automated exam seat allocation system using core Data Structures and Algorithms concepts. The website provides an interactive simulation of the allocation process, while the main algorithm is implemented in C++ for academic evaluation.

How This Works

• Students are allocated classroom-wise and time-slot-wise
• Question paper sets (A/B/C/D) are assigned cyclically
• Subject conflicts and per-classroom limits are enforced
• Late students are handled using a FIFO queue
• Invigilators are assigned using load-balanced scheduling
